Что такое модули?

Модули позволяют включать и выключать отдельные функции бота без изменения кода. Каждый модуль можно настроить индивидуально для этого бота.

Доступные модули:

  • registration — регистрация пользователей
  • receipts — загрузка и проверка чеков
  • user — профиль, FAQ, поддержка
Внимание: Изменения применяются после перезапуска бота.
